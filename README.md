# Ecommerce Website

A simple ecommerce website built with HTML, CSS, and JavaScript.

## Features
- Product listing
- Add to cart
- Search and filter
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository  
   `git clone https://github.com/pv052k01/Ecommerce-Website.git`
2. Open the project folder
3. Open `index.html` in your browser
